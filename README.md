# The Prist Nutri Co. - Premium E-Commerce Website

A modern, production-ready premium e-commerce website for The Prist Nutri Co., built with HTML5, CSS3, and Vanilla JavaScript.

## 🌟 Features

### Shopping Experience
- **Product Catalog**: 10 premium nutri products with detailed information
- **Advanced Search & Filter**: Search by name and filter by category
- **Smart Sorting**: Sort by price, newest, best-sellers
- **Wishlist**: Save favorite products
- **Shopping Cart**: Persistent cart with LocalStorage
- **Multiple Weight Options**: 100g, 250g, 500g, 1kg with dynamic pricing

### Product Information
- Multiple product images
- Detailed descriptions
- Ingredients list
- Nutrition information table
- Health benefits
- Customer ratings and reviews
- Related products
- Frequently bought together

### Checkout & Payment
- User authentication system
- Secure checkout form
- Multiple payment options:
  - Cash on Delivery (COD)
  - Dummy Online Payment
  - WhatsApp Checkout
- Automatic WhatsApp order message generation
- Order summary with calculations

### User Features
- Dark Mode Toggle
- Recently Viewed Products
- Newsletter Subscription
- Customer Reviews
- FAQ Section
- Combo Builder
- Back to Top Button
- Loading Spinner

### Design & Performance
- **Premium UI**: Glassmorphism design with smooth animations
- **Color Scheme**: Green & White theme
- **Responsive Design**: Mobile-first approach
- **Accessibility**: WCAG compliant, semantic HTML
- **Performance**: Lazy loading, optimized assets
- **SEO**: Meta tags, Open Graph, Structured data

## 🛠 Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Flexbox, Grid, Custom properties
- **JavaScript**: Vanilla ES6+
- **LocalStorage**: Client-side data persistence
- **No External Frameworks**: Pure vanilla implementation
